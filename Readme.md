# DATABASE SYSTEMS PROJECT

## Course Information

**Course:** Database Systems

**Project Topic:** Database Design and SQL Implementation

---

# Project Overview

This project consists of two independent database systems developed using MySQL.

The objective of the project is to apply database design principles, normalization techniques, integrity constraints, indexing strategies, and SQL query development to solve real-world management problems.

The project includes:

* Database schema design
* Primary Key and Foreign Key implementation
* CHECK and UNIQUE constraints
* Sample data population
* Index creation
* View creation
* Business and analytical SQL queries

---

# Project 1: TinyCollege

## Description

### Main Functional Areas

* School Management
* Department Management
* Professor Management
* Student Management
* Course Management
* Class Scheduling
* Enrollment Management
* Academic Performance Analysis

* DEPARTMENT
* PROFESSOR
* COURSE
* SEMESTER
* BUILDING
* ROOM
* CLASS
* STUDENT
* ENROLL

* Referential Integrity

# Database Design Principles

The databases were designed according to relational database design principles.

### Integrity Constraints

* Primary Keys
* Foreign Keys
* Unique Constraints
* Check Constraints

### Normalization

The database structures follow:

* First Normal Form (1NF)
* Second Normal Form (2NF)
* Third Normal Form (3NF)

### Optimization

* Indexes on frequently queried columns
* Optimized JOIN operations
* Aggregate and analytical queries
* Reporting Views
